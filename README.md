# バーコード生成アプリ

## 概要
このアプリケーションは、複数のバーコードを一括で生成し、PNG形式またはSVG形式で保存できるデスクトップアプリケーションです。

![image](https://github.com/user-attachments/assets/5705de17-06fe-4bb2-991c-f3f62cf12f2a)


## 機能
- 複数のバーコード同時生成
- 対応フォーマット：
  - JANコード（EAN-13）
  - ITFコード
  - GS1データバー
- PNG/SVG形式での保存対応
- バーコードタイプの一括変更
- 入力値の一括クリア

## 使用方法
1. アプリを起動
2. バーコード番号を入力
3. バーコードタイプを選択
4. 「バーコードを作成」をクリック
5. 必要に応じてPNGまたはSVG形式で保存
